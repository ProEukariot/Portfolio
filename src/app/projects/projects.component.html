ToDo App

The Sticky Notes Todo app is a web application I developed this using Angular to help users organize tasks visually. Users can create, edit, and delete sticky notes, with features like drag-and-drop and layering for prioritization.

I crafted a dynamic and responsive frontend that makes adding, editing and organizing notes a breeze. The app leverages Angular's powerful features to keep everything running smoothly and efficiently. Styling the app with SCSS allowed me to add a nice visual style and keep the interface clean and modern.

TypeScript was my choice for adding type safety and making the codebase reliable and maintainable. This made it easier to find bugs and made the development process much more enjoyable.

This project is a great example of how Angular can be used to create a practical, everyday tool. It's all about combining functionality with a little fun and style. Give it a try and see how it can help you manage your tasks!

Core features

  • Drag and Drop: Users can easily rearrange sticky notes by dragging and dropping them on the board.
  • Sticky Notes Editing: Users can create, delete, and edit sticky notes, enhancing interactivity and flexibility.
  • Interactive Layering System: The system dynamically adjusts the z-index to ensure that the last interacted note appears on top.

Challenges

  • Drag and Drop Implementation: Developed custom Angular directives for a smooth drag-and-drop experience, managing DOM events and real-time state updates.
  • Interactive Layering System: Implemented dynamic z-index adjustments to manage layering and visibility of sticky notes, ensuring that user interactions are accurately reflected.

3D marketplace

Welcome to my latest development, a dynamic 3D marketplace application that reimagines the way users interact with digital assets. This project is built using a robust technology stack that includes Angular and TypeScript at its core, leveraging the full set of features these frameworks offer for seamless development.

I integrated Material UI and TailwindCSS to create an elegant and responsive UI, ensuring that the visuals are as captivating as the 3D models themselves. Speaking of 3D, I used ThreeJS to create an immersive, interactive dimension in the marketplace, allowing users to explore products in a truly immersive way.

On the backend, NestJS and TypeORM provide a solid foundation, while SQL Server takes care of the database needs. Authentication is secure, keeping user data safe, and payment processing is a breeze thanks to Stripe integration. To make the application scalable and efficient, Azure blob storage is used to handle all media assets.

I didn't stop there. The app has advanced filtering options that allow users to easily find exactly what they are looking for. To keep things running smoothly, RxJS is used to manage asynchronous data streams.

This project demonstrates my ability to combine different technologies to create a functional, user-friendly application that is both visually impressive and technically sound. Check out the program and experience the future of 3D markets!

Core features

  • 3D Model Viewer: Integrated Three.js for an interactive 3D model viewer, allowing users to view, rotate, and scale models.
  • Payment Processing: Ensured PCI DSS compliance and integrated Stripe API for secure payment transactions.
  • Blob Storage Management: Utilized Azure Blob Storage for efficient file handling and fast access to large 3D models.
  • Authentication and Authorization: Developed secure user authentication with registration and login functionalities.
  • Product Sorting and Filtering: Implemented responsive sorting and filtering for quick and effective product discovery.
  • Advanced Search Functionality: Enabled searching by keywords and tags with real-time suggestions.
  • Responsive and Cross-Browser Compatibility: Ensured seamless experience across different devices and browsers.

Challenges

  • 3D Model Viewer Optimization: Leveraged Three.js and lazy loading to provide a smooth, cross-device 3D experience.
  • Secure Payment Integration: Integrated Stripe with server-side processing and real-time transaction management.
  • Managing Large Files: Implemented asynchronous operations with Azure Blob Storage for efficient handling of large 3D files.
  • Authentication System: Developed a scalable authentication system using JWT tokens.
  • Optimizing Sorting and Filtering: Applied indexing and query optimization techniques to handle large datasets efficiently.
